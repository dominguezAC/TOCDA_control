CONTROLLER DESING - OPTIMIZATION

Now the simulink model is modified to return the integra of an error function in order to
optimize this integral with the values of Kp, Kd and Ki. As before a simulation is made
from the triming point in equilibrium, setting a commanded angular speed of 45 rad/s
and at t=5 a step in wind speed is introduced (from 10 to 11 m/s).
The function used for this optimization was:
∫ [∆u^2 * K1 + ∆ω^2 * K2 + (∂∆ω/∂t)^2 * K3 ] dt
With:
K1 = 50
K2 = 1000
K3 = 5
this method is based on the same foundation of the lqr method in MIMO systems
BUT done with a non linear modell and numerically instead of analiticaly.
